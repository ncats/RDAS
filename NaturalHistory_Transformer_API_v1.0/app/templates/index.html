Welcome to the Article Prediction API!

This application demonstrates a simple API for text classification using Flask and Transformers.

Get Started

You can make predictions by accessing the /predict endpoint. This endpoint accepts both GET and POST requests.

  • Use a GET request for simple tests via the browser: Try it now
  • Use a POST request for more complex queries. This requires tools like curl, Postman, or similar.

More Information

Check out the GitHub repository for more details on how to use and extend this API.
